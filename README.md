# chatroom

## Features
 - Use react.js
 - Use webpack
 - Use node.js

## Install
```bash
git clone <reop>
```

## Commands
 - Before development
```
npm install
```

 - Development
```
npm start
```
 - Build to package will create `dist` directory
```
npm run build
```
 - Production
```
npm install -g serve
```
 - Start 
```
serve -s build
```
